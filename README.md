## Overview

Unit Converter Tool is a Python-based desktop application developed using Tkinter. The application enables users to perform quick and accurate conversions between different units of Length, Weight, and Temperature through an interactive graphical user interface.

The project was created to demonstrate Python programming concepts, GUI development, event handling, and user input validation.

---

## Features

* Length Conversion

  * Meter ↔ Kilometer
  * Meter ↔ Centimeter
  * Kilometer ↔ Centimeter

* Weight Conversion

  * Gram ↔ Kilogram

* Temperature Conversion

  * Celsius ↔ Fahrenheit

* User-Friendly Graphical Interface

* Accurate Unit Conversion

* Error Handling for Invalid Inputs

* Fast and Efficient Performance
